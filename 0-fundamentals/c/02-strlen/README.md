# Задача 02: Собственный strlen на C

## Описание
Реализовать функцию подсчета длины строки без использования стандартной strlen.

## Цель
Понять работу с указателями и строками в C.

## Требования
- Функция должна принимать `const char*`
- Возвращать `size_t`
- Не использовать стандартные библиотечные функции

## Как запускать
```bash
gcc src/strlen.c -o strlen_test
./strlen_test
